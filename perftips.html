Making Netflix load faster with Flix Plus


The Netflix homepage has around 40 sections and 1500 posters on it. Some Flix Plus capabilities--primarily those that update each section or poster--can make Netflix take a bit longer to load. Disabling certain scripts can help, if you're willing to sacrifice those features. These are the scripts that can slow things down:

  • Hide sections and Remove scrollbars add a button to each section, which can slightly increase page load time.
  • Fade/tint/hide rated and Fade/tint/hide watched have two parts to them:
    • Search the website for the existence of each poster id that was previously stored and update the site to fade/tint/hide it.
    • Every week, it requests updated data. This takes about a hundredth of second for every show or episode you have rated or watched. You do not need to wait for this to finish, but the data will not be updated until you do.
  • Fade/tint/hide duplicates looks at every poster on the page, keeps track of the posters it has seen, and hides it if it has seen it before and it is not in My List.
  • Whenever a script makes it so that a poster is visible on page load that otherwise wouldn't be (Remove scrollbars and hiding rated, watched, duplicates do this), Flix Plus needs to check whether or not each visible poster is loaded and then load it if not. If you disable all such functionality, this check will be skipped to give a significant speed boost.
  • Clicking on poster shows details instead of playing changes the play link for every poster and removes the play button from on top of the poster.
  • Add keyboard shortcuts gets a list of sections and the posters in each section. If you disable navigation (to use keyboard shortcuts just for controlling the player and jumping around pages), it loads very fast.

You can speed up the My List page in manual mode by disabling Allow rating by half star, Show box images, and Clicking on poster shows details instead of playing since these scripts update every entry.

The large majority of scripts (those not listed here) load very quickly (< 10 ms). The following buttons will automatically disable the features listed above for optimal speed.
